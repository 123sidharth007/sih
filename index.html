Welcome to Revivers

This is a distraction-free web player and admin platform that serves the same educational video content (YouTube) but with a minimal UI, teacher controls and enforced playlists .

Students waste time on YouTube because of distractions like recommendations, ads, and comments.

Our solution, EduFocus, is a distraction-free website that plays YouTube videos in a clean, focused interface

Teachers can create playlists, lock controls, and track progress through an admin dashboard

A Raspberry Pi gateway or browser extension redirects YouTube links to our platform, keeping learners focused

This ensures productive, distraction-free learning aligned with the Smart Education theme
